Whook Testimonials is a WordPress plugin which is basically designed for the providing the testimonial's. In this plugin there are various options available by which user can share the experiences.

=== Description ===

<strong>Whook Testimonials</strong> is a plugin to use the feedback form on your <strong>WordPress</strong> website. It is a user friendly and feature rich plugin to add a responsive <strong>Testimonials</strong> review or <strong>star ratings</strong> on website.

It is a simple and easy to customize plugin to the pages on website. There are multiple feedbacks styles available, you currently need to go to backend select appropriate styles. Only use instance of shortcode on the post or page of website.

=== Features ===
1. No Flash Needed
1. Compatible with chrome, Firefox, IE8, IE9, Safari
1. You can select any style pattern
1. Support star rating 
1. You can select attributes of the feedback form and also make them required
1. You can implement Google Recaptcha to your feedback form
1. There are various option with the question ratings like numeric and alphabetic both
1. You can sort the ratings

=== How to use ===

= Shortcode =
1.	[whook-test-form] 
     (Recommended)
    <strong>Paste shrotcode where you want on your website</strong>

1.	[whook-test-feedback-show layout=1 limit=10]
     <strong>Using this shortcode admin can change the layout, in this  user information and image both show after feedback.

 
1.	[whook-test-feedback-show layout=2 limit=10]
     <strong>Using this shortcode also admin can change the layout, in this  user information show in a list format after feedback.
